# Thinking-Time Feedback

Meta project for user feedback on any Thinking-Time skill. Channels feedback from Teams, KIO Champions and KIO Implementer as pre-filled GitHub issues to kio-rocks/kio-thinking-time. Covers all modules (strategy-canvas and future modules) — install once next to your workshop project(s).

## Einrichtung

1. In Claude ein neues Projekt anlegen
2. `projektanweisung.md` als Projektanweisung einfügen
3. Alle `.skill`-Dateien unter *Fähigkeit hochladen* hinzufügen

Claude führt ab hier durch den Rest.

## Skills

| Skill | Was er tut |
|-------|-----------|
| `feedback` | Turns user feedback about any Thinking-Time skill into a pre-filled GitHub issue URL for kio-rocks/kio-thinking-time. No auth, no tokens — the user clicks the link, GitHub opens a form with all fields filled, the user submits.
