## Was ist kio-thinking-time?

Inhabergeführter Mittelstand nutzt KI bereits — aber ohne messbare Wirkung. Der Engpass ist nicht das Tool. Der Engpass ist Klarheit.

**KIO** ist ein Führungssystem, das zuerst Klarheit herstellt und dann KI als Verstärker einsetzt. Thinking Time ist die Kern-Methodik: strukturierte Denkzeit, die ungeklärte Komplexität in verteilbare Klarheit übersetzt.

Dieses Repository stellt die ausführbaren Skills für Claude Desktop bereit — als Vorlagen, die direkt in realen Führungs-Sitzungen eingesetzt werden.

## Setup — Claude Desktop

1. **Claude Desktop öffnen** — [claude.ai/download](https://claude.ai/download)
2. **Neues Projekt erstellen** — Klick auf „+ New Project" in der linken Sidebar
3. **Projektanweisung einfügen** — `strategy-canvas/projektanweisung.md` in das Feld „Project Instructions" kopieren
4. **Knowledge hochladen** — die ZIPs aus `strategy-canvas/dist/` ins Projekt hochladen
5. **Sitzung starten** — im Chat mit dem Skill-Namen beginnen (z.B. „Starte Core Competency")
