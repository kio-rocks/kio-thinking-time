## Einrichtung

1. Gewünschtes Paket herunterladen und entpacken
2. **Claude Pro:** Unter *Anpassen* → *Skills* hochladen oder ersetzen
3. **Claude Team:** Unter *Organisationseinstellungen* → *Skills* hochladen oder ersetzen
