# KIO Development Index

Globaler Routing- und Übersichts-Skill für alle KIO Dev-Hüllen. Beantwortet "wo arbeiten wir gerade dran?", "in welche Hülle gehört diese Änderung?" und "sind unsere Validierungs-Subjekte konsistent?" ohne dass die einzelnen Dev-Hüllen einzeln konsultiert werden müssen.

## Einrichtung

1. In Claude ein neues Projekt anlegen
2. `projektanweisung.md` als Projektanweisung einfügen
3. Alle `.skill`-Dateien unter *Fähigkeit hochladen* hinzufügen

Claude führt ab hier durch den Rest.
